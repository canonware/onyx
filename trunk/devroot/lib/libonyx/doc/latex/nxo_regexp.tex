\subsection{nxo\_regexp}
\label{nxo_regexp}
\index{nxo_regexp@\classname{nxo\_regexp}{}}

The \classname{nxo\_regexp} class is a subclass of the \classname{nxo} class.

a\_insensitive, cw\_bool\_t a\_overlap}}
