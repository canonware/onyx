\subsection{gcdict}
\label{gcdict}
\index{gcdict@\classname{gcdict}{}}

The \classname{gcdict} functions implement the operators contained in
\onyxop{}{gcdict}{}.  Only the C API is documented here; see
Section~\ref{sec:gcdict} for operator semantics.
